# Portfólio Pessoal - Iago Dantas

Este é o repositório do meu portfólio pessoal, desenvolvido com o objetivo de apresentar minha trajetória profissional, habilidades técnicas, projetos e evolução durante minha transição de carreira para a área de tecnologia.

## Sobre o projeto

O portfólio tem como principal objetivo destacar:

- Resumo profissional e acadêmico;
- Projetos desenvolvidos em Java, Spring Boot, HTML, CSS, JavaScript e React;
- Hard skills e soft skills;
- Contatos e redes sociais;
- Interface moderna, responsiva e de fácil navegação;
- Evolução técnica durante minha jornada como desenvolvedor.

## Tecnologias utilizadas

### Front-end

- React
- JavaScript
- HTML5
- CSS3
- Bootstrap 5
- React Bootstrap

### Ferramentas

- Git
- GitHub
- Vite
- VS Code

## Funcionalidades atuais

- Estrutura inicial em React;
- Componentização de seções como Header e Hero;
- Navegação por âncoras;
- Botão de contato via WhatsApp;
- Download de currículo;
- Estilização com variáveis CSS;
- Layout inicial responsivo.

## Funcionalidades previstas

- Seção "Sobre mim" aprimorada;
- Seção de projetos com cards dinâmicos;
- Seção de skills técnicas e comportamentais;
- Seção de contato funcional;
- Responsividade aprimorada para mobile;
- Animações leves na interface;
- Melhorias de acessibilidade;
- Possível integração futura com backend/API para gerenciar projetos dinamicamente;
- Deploy na Vercel ou GitHub Pages.


## Objetivo

Este projeto faz parte da minha evolução prática como desenvolvedor, aplicando conceitos de componentização, organização de código, responsividade, boas práticas de front-end e construção de uma presença profissional online.
